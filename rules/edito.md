# Règles éditoriales

Ce document synthétise les règles éditoriales actuellement utilisées sur le site de La Cueillette du Geai.

## Positionnement

- La marque se présente comme une `herboristerie paysanne`.
- L’ancrage géographique doit être explicite : `Biziat`, `Ain`.
- Le site vend une activité réelle, locale et artisanale, pas un univers fictif ou conceptuel.
- Le ton doit rester simple, précis, crédible et incarné.
- Le site n’est pas une boutique en ligne. Cela doit être clair quand on parle d’achat.

## Promesse de première vue

La première vue doit permettre de comprendre en quelques secondes :

- ce qui est proposé : `tisanes artisanales` et `sirops de plantes`
- où cela se passe : `Biziat`, `Ain`
- ce qui rend l’activité singulière : `cueillettes locales`, `petites séries`, `transformation artisanale`

Le hero doit contenir proprement :

- le nom de la marque
- un `h1` clair et court
- un texte d’appui informatif
- une adresse ou point de vente visible
- un CTA principal

## Ton et style

- Éviter le ton publicitaire forcé.
- Éviter les formulations creuses de type “expérience unique”, “passion”, “authenticité” si elles ne disent rien de concret.
- Préférer les formulations sobres, agricoles, artisanales et ancrées dans le réel.
- Faire parler le produit, la méthode, le territoire et le geste.
- Le texte doit rester vendeur, mais sans sur-promesse.

## SEO rédactionnel

Les mots-clés doivent être intégrés naturellement, sans surcharge.

Expressions à faire vivre dans le contenu :

- `herboristerie paysanne`
- `tisanes artisanales`
- `sirops de plantes`
- `cueillettes locales`
- `cueillette locale`
- `plantes cueillies localement`
- `plantes séchées naturellement`
- `production locale`
- `petites séries`
- `Biziat`
- `Ain`

Le SEO ne doit jamais dégrader la lisibilité ni donner un texte mécanique.

## Hiérarchie éditoriale

- `H1` : message principal de la page, unique, clair, orienté activité + produit.
- `H2` : titres de section, lisibles, éditoriaux, non grandiloquents.
- `H3` : sous-parties, cartes produits, cartes méthode, noms de plantes.
- Le paragraphe d’introduction d’une section doit rester court.
- Les sections doivent éviter les redondances entre titre, intro et listes.

## Longueur et densité

- Le hero ne doit pas être bavard.
- Une introduction de section doit idéalement tenir en 2 ou 3 phrases courtes.
- Les cartes doivent contenir une idée claire chacune.
- Éviter les blocs trop abstraits ou trop institutionnels.
- Réduire les répétitions entre sections quand elles disent déjà la même chose.

## Structure de contenu actuelle

La structure éditoriale de référence est :

1. `Hero`
2. `À propos`
3. `Produits`
4. `Méthode`
5. `Acheter sur place`
6. `Plantes`
7. `Contact`

Chaque section doit avoir une fonction claire :

- `Hero` : comprendre l’offre et agir
- `À propos` : situer la marque et sa démarche
- `Produits` : expliquer ce qui est vendu
- `Méthode` : crédibiliser la pratique
- `Acheter sur place` : clarifier le mode d’achat
- `Plantes` : montrer la palette végétale travaillée
- `Contact` : convertir vers un contact ou une visite

## Règles sur les produits et plantes

- Ne parler que des plantes et parfums réellement présents dans les assets ou effectivement montrés sur le site.
- À ce jour, la liste de référence est :
  - `Camomille`
  - `Menthe aquatique`
  - `Menthe poivrée`
  - `Reine des prés`
  - `Sureau`
  - `Thym citron`
  - `Verveine citron`
- Éviter les listes ouvertes du type `autres plantes` si elles ne sont pas documentées.

## Règles de crédibilité

- Toujours privilégier le concret :
  - lieu
  - geste
  - méthode
  - formats
  - point de vente
  - contact
- Les prix doivent être lisibles et formulés clairement.
- Les visuels réels sont prioritaires sur les visuels décoratifs quand ils servent la preuve.
- Les éléments de réassurance doivent rester simples : local, artisanal, petite série, contact direct.

## CTA et conversion

- Une action principale doit ressortir dans la première vue.
- Les actions secondaires doivent rester discrètes.
- Les CTA doivent être courts, explicites et liés à l’action réelle :
  - `Découvrir les produits`
  - `Voir le point de vente`
  - `Envoyer un email`
- Ne pas employer de CTA trompeurs de type achat immédiat si le site n’est pas e-commerce.

## Ce qu’il faut éviter

- Le ton “maquette de marque” ou “marque lifestyle”.
- Les phrases trop longues dans le hero.
- Les répétitions d’adresse ou d’information pratique.
- Les formulations vagues comme `univers`, `expérience`, `authentique` sans précision.
- Les blocs de texte trop denses pour une landing page.
- Les listes de plantes non sourcées par les contenus réels du site.

## Critère de validation éditoriale

Une page est considérée comme alignée si :

- la marque, l’activité et le lieu sont compris immédiatement
- le visiteur sait ce qui est vendu
- le visiteur comprend que la vente se fait sur place ou par contact direct
- le ton paraît professionnel, agricole et artisanal
- le texte reste lisible, non redondant et non surchargé SEO
